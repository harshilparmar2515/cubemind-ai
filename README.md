# CubeMind AI

An advanced 3D Rubik's Cube simulator built with React, Three.js, and AI-powered solving assistance.

## Features

- Realistic 3D Rubik's Cube
- Cube State Engine
- Smart Scramble Generator
- AI Learning Assistant
- Solve Analysis
- Multiplayer Challenges
- Statistics Dashboard
- Firebase Integration

## Tech Stack

- React
- Vite
- Three.js
- React Three Fiber
- Tailwind CSS
- Firebase
- OpenAI API
